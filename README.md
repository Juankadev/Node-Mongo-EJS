# Node-Mongo-EJS
Sitio web para mostrar frutas recibidas desde MongoDB

localhost:3008/

Antes hay que:

Ejecutar npm i

Configurar un archivo .env con PORT=3008, y MONGODB_URL = "url de cluster"

Importar en el cluster el archivo frutas.json
